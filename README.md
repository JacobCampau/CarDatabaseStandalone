<h1>Car Database (Standalone)</h1>

<h2>What is this project?</h2>

A standalone version of my [Original Car Database](https://github.com/JacobCampau/CarDatabase.git) project. The original project required the user to set values in the code, download the dataset, and run the PostgreSQL on their device before running the application. To make this a more accessable tool, this project ensures there is no such expectation from the user.
